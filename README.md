Nexora Store

Project Description

Nexora Store is a modern and responsive e-commerce web application built with React. It allows users to browse products, search and filter items, view product details, and manage a shopping cart with a smooth and user-friendly interface.

---

Features

- Browse products with grid and list view options
- Search products by name
- Filter products by category
- Sort products by price and rating
- View detailed product information
- Add, remove, and update items in the cart
- Persistent cart using localStorage
- Responsive design for mobile and desktop
- Dark mode / Light mode toggle
- Checkout process with multi-step form
- Notifications (toast messages) for user actions
- Fast data fetching with react Query
- Pagination system

---

🛠️ Tools & Libraries Used

- React
- Vite
- Material UI (MUI)
- Context API
- Redux Toolkit
- React Router DOM
- React Query (TanStack Query)
- Axios
- Notistack (for notifications)

How to Run the Project

1. Clone the repository:

git clone <your-repo-link>

2. Navigate into the project folder:

cd product-store-app

3. Install dependencies:

npm install

4. Run the development server:

npm run dev

5. Open your browser and go to:

http://localhost:5173
